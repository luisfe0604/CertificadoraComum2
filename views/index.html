Uma visão geral dos tópicos abordados neste guia de estudos de POO

Introdução a POO

POO é um paradigma que organiza o código em objetos interativos, proporcionando modularidade, flexibilidade e reutilização.

Alocação de memória

Processo de solicitar/utilizar memória durante o processo de execução de um programa


Encapsulamento

É a técnica utilizada para esconder uma ideia, ou seja, não expôr detalhes internos para o usuário

Reflexividade

Consiste na capacidade que um programa tem de examinar e modificar sua própria estrutura ou comportamento

Polimorfismo

Permite que referências de tipos de classes mais abstratas representem o comportamento das classes concretas

Herança

Tem o mesmo significado para o mundo real. Como um filho pode herdar alguma característica do pai, é possível que uma classe herde atributos e métodos da outra

Acoplagem e Coesão

Coesão e Acoplamento são princípios de engenharia de software
